# Order Fuel

A fuel delivery application built with React, TypeScript, and Vite.

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Run Tests
```bash
npm test
```

## Project Structure

- `/src/pages` - Application pages
- `/src/components` - Reusable components
- `/src/contexts` - React contexts (Auth, etc.)
- `/src/lib` - Utilities and types
- `/src/hooks` - Custom React hooks

## Features

- User authentication
- Fuel ordering system
- Order management
- Delivery tracking
- Analytics dashboard
- Multi-role support (Customer, Admin, Manager, Attendant, Rider)
